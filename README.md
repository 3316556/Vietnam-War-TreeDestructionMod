# Tree Destruction Mod

Adds tree destruction interactions for explosions, napalm/fire effects, and related battlefield damage.

## Install

1. Install MelonLoader 0.7.2 (Open-Beta) for the IL2CPP Windows build of *Vietnam War* (Unity 2020.3.30f1, Game Version 0.1).
2. Download the release ZIP for this mod.
3. Extract it into your game folder, usually SteamLibrary\steamapps\common\VietnamWar.
4. Confirm that Mods\TreeDestructionMod.dll exists in the game folder.
5. Launch the game.

To uninstall, delete Mods\TreeDestructionMod.dll from the game folder.

## Contents

This repository intentionally ships only the compiled mod DLL and install/config documentation. It does not include game files, game assets, MelonLoader, decompiled source, or mod source code.

## Compatibility

This game is extremely version-specific. These mods were built and tested against this exact environment — match it:

| Component | Version |
| --- | --- |
| Game | Vietnam War (ThoriumGameLab), Game Version 0.1 |
| Unity | 2020.3.30f1 (IL2CPP, x64) |
| MelonLoader | 0.7.2 (Open-Beta), net6 runtime |

No additional libraries are required — UniverseLib is **not** needed by this mod. A different game build or MelonLoader version will likely require a rebuilt DLL.
